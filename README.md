# ghDS - Template project


# Start dev environment

* Requirements
    * node v10.13.0

1. Install dependencies - on project root folder:
    ~~~~
    npm install
    npm run install-all
    ~~~~

2. Start from project root folder:
    ~~~~
    npm start
    ~~~~

3. Or if you want to start separately.  
   First start Server:
    ~~~~
    cd server
    node server.js
    ~~~~

4. And then start Client:
    ~~~~
    cd client
    npm start
    ~~~~
